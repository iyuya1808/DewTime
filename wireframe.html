主要フロー：
① TimerView (未スタート) → [スタートボタン] → StartSheet → [スタート！] → TimerView (実行中)
② TimerView (実行中) → [いってきます！] → DepartureConfirmView → [確認] → DepartureResultView → [また明日！] → TimerView (リセット)
③ TimerView → [植物カード] → PlantPickerSheet → 植物選択 → 閉じる
④ GardenView → [日付セル] → WateringRecordDetailSheet
⑤ GardenView → [📅カレンダー] → MonthlyGardenView
⑥ CollectionView → [種族カード] → SpeciesDetailSheet → [花タイル] → FlowerDetailSheet
⑦ SettingsView → [スケジュール行] → RoutineEditorView → [タスク追加] → AddRoutineItemSheet
⑧ SettingsView → [データ管理] → DataManagementView
